## 技術カンファレンスとリーダーによる評価

### performance.now()
パフォーマンス特化の国際会議で「Webサイト構築の模範」として選出。JavaScriptを最小化するIslandアーキテクチャが、Lighthouse満点を実現する標準手法として認知。

### ViteConf
Viteエコシステムの筆頭フレームワークとして登壇。Vite 6 の開発段階から協力し、Environment API の仕様策定と実装をリード。

### Kent C. Dodds
Remix/Reactの世界的権威が「コンテンツ駆動サイトにおける最良の選択肢」と公言。Epic Web等の教育プロジェクトにおいて、大規模コンテンツ管理の推奨スタックに採用。

## Frontmatterとclient scriptの境界

サーバーサイドで実行される領域と、ブラウザで実行される領域には...

たとえば...

```astro
---
// frontmatter領域
import { Image } from 'astro:assets';
const { title, description } = Astro.props;

console.log('server-side script executed');
---
// 省略
<script>
// client script領域

console.log('client-side script executed');
</script>
```
